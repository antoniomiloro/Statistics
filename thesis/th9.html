Th9: The Poisson Process: Meaning, Properties, Simulations

The Poisson Process is a type of stochastic process, often used to model random events such as the arrival of customers at a shop or emails to a mail server.
A Poisson process is a renewal process where the interarrival intervals have an exponential distribution function.

Properties:

- The number of events in any interval of length t is a Poisson random variable with parameter λt, where λ is the rate of the process.
This means that the probability of observing k events in an interval of length t is given by:
\[P(N(t) = k) = \frac{(\lambda t)^k e^{-\lambda t}}{k!}\]

- The distribution of the number of arrivals between, say, t and t+s depends only on the length of the interval s and not on the starting point t.
- All inter-arrival times are exponentially distributed and independent with mean 1/λ.
This means that the probability that the time between two events is greater than t is given by:
\[P(X > t) = e^{-\lambda t}\]

- An alternative definition of a Poisson process is of a counting process whose inter-arrival times are distributed exponentially and independently.
- The Poisson process is unique among renewal processes due to the memoryless property of the exponential distribution.

Simulations:

There are several methods to simulate a Poisson process, including direct methods based on the homogeneous Poisson stochastic process having exponential inter-arrival times, inversion methods that search through a table of cumulative Poisson probabilities,
methods that use the recursive properties of the Poisson distribution, and acceptance-rejection methods.
Simulating a Poisson point process on a computer is usually done in a bounded region of space, known as a simulation window, and requires two steps: appropriately creating a random number of points and then suitably placing the points in a random manner.
There are also demonstrations available that show simulated paths of the Poisson process.

Examples:

- Simulating the number of customers arriving at a store in a given hour.
We can assume that the arrival rate is λ = 10 customers per hour, and generate random numbers from a Poisson distribution with mean λt, where t is the length of the interval in hours.
For example, if we want to simulate the number of customers in the first 15 minutes, we can use t = 0.25 and generate a random number from a Poisson distribution with mean 10 × 0.25 = 2.5.
This can be done using the rpois function in R, as shown below:
# Set the seed for reproducibility
set.seed(123)

# Simulate the number of customers in the first 15 minutes
rpois(1, lambda = 10 * 0.25)
#> [1] 1

- Simulating the time of the first earthquake in a region.
We can assume that the earthquakes follow a Poisson process with rate λ = 0.01 earthquakes per day, and generate random numbers from an exponential distribution with mean 1/λ.
For example, if we want to simulate the time of the first earthquake, we can generate a random number from an exponential distribution with mean 1/0.01 = 100 days.
This can be done using the rexp function in R, as shown below:
# Set the seed for reproducibility
set.seed(123)

# Simulate the time of the first earthquake
rexp(1, rate = 0.01)
#> [1] 52.2019
